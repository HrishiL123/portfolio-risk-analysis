# Portfolio Risk Analyzer

A Python-based tool that fetches real-time stock data, calculates portfolio returns, volatility, and Sharpe ratio, and visualizes riskâ€“return insights. 



## Features

- Fetch historical stock data using `yfinance`
- Calculate **annualized returns**, **volatility**, and **Sharpe ratio**
- Generate **price trends** and **risk vs return charts**
- Supports **single or multiple tickers**
- Optional: analyze **portfolio-level metrics**



## Tech Stack

- Python 3.14  
- Libraries: `yfinance`, `pandas`, `numpy`, `matplotlib`, `scipy`
